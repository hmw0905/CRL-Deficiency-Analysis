Project Overview

Turning unstructured FDA letters into analysis-ready data

The Problem

FDA Complete Response Letters explain why a drug or biologic application could not be approved in its current form. Because these letters are narrative documents, systematic large-scale analysis is difficult.

The Solution

I manually coded publicly available CRLs into structured datasets, then used SAS to summarize trends, test associations, model outcomes, and create a dashboard for interactive reporting.

Nearly 40% of CRLs involved multiple deficiency domains.
